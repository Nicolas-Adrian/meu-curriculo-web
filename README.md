# Currículo Web - Nicolas Adrian

Este projeto é um **currículo online desenvolvido com HTML e CSS**, criado com o objetivo de apresentar minhas informações acadêmicas e profissionais em formato de página web e também praticar conceitos básicos de desenvolvimento front-end.

## 📄 Sobre o Projeto

A página funciona como um **currículo digital**, contendo informações essenciais como:

* Nome e informações de contato
* Localização
* Links para GitHub e LinkedIn
* Formação acadêmica
* Cursos complementares

O projeto foi desenvolvido utilizando apenas **HTML e CSS**, focando na construção da estrutura da página e na estilização básica do conteúdo.

## 🛠 Tecnologias Utilizadas

* **HTML5** – responsável pela estrutura da página
* **CSS3** – responsável pela estilização e aparência visual do site

## 📂 Estrutura do Projeto

```text
/
├── index.html
└── style.css
```

* **index.html** → contém a estrutura e o conteúdo do currículo
* **style.css** → contém as regras de estilização da página

## 🎯 Objetivo do Projeto

Este projeto foi criado com alguns objetivos principais:

* Praticar conceitos básicos de **HTML e CSS**
* Aprender a estruturar páginas web
* Criar um **currículo online**
* Iniciar um **portfólio no GitHub**
